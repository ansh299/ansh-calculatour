# 🚀 ANSH Project: Ultimate Power Calculator

A high-performance scientific calculator app combining premium UI design with an advanced computing engine. 

### 🌟 Highlights
- **Dual Modes:** Quick-switch standard and scientific layout panels.
- **Micro-Animations:** Fluid button-press scaling and custom pointer canvas ripple waves.
- **Precision Engine:** Built-in base-10 math stabilizer that eliminates JavaScript floating-point errors (e.g., $0.1 + 0.2 = 0.3$).
- **Smart Security:** Dynamic evaluation inside safe sandbox execution environments rather than risky `eval()`.
- **Zero Dependencies:** Formed using raw HTML5, CSS3 Grid/Flexbox, and Vanilla ES6+ JavaScript.
- # 🧮 ANSH Project - Glassmorphic Scientific Engine

An architectural exploration into creating zero-dependency web utilities with a focus on UI/UX responsiveness and secure client-side mathematics evaluation.

### 🛠️ Core Capabilities
- **Advanced Trigonometry & Logs:** Full support for `sin`, `cos`, `tan`, `log`, `ln`, along with dynamic variables for Pi (`π`) and Euler's Number (`e`).
- **Input Sanitization Logic:** Under-the-hood checks intercept consecutive operators and double decimals, keeping the UI breakdown-proof.
- **Glassmorphism Theme:** High-performance CSS background blurring with custom color-coded button categories (Actions vs. Operators).
- **Responsive PWA Framework:** Formatted to perfectly scale from high-resolution desktop environments down to mobile viewports.
